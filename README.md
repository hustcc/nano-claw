## About nanobot

nanobot is an ultra-lightweight personal AI assistant that delivers core agent functionality in just ~4,000 lines of code - 99% smaller than Clawdbot's 430k+ lines.

### What's Inside
- Multi-LLM provider support (OpenRouter, Anthropic, OpenAI, vLLM, etc.)
- 9+ communication channels (Telegram, Discord, WhatsApp, Email, etc.)
- Built-in tools (Shell, File Operations, Spawn sub-agents)
- Scheduled tasks with Cron support
- Skills system for extensibility
- Voice transcription support
