# Weather Information

Get current weather information for any location.

## Usage

Ask me about the weather in any city or location, for example:
- "What's the weather in New York?"
- "Tell me the weather forecast for London"
- "Is it raining in Tokyo?"

## Data Source

Weather data is fetched from public weather APIs.

## Examples

**User**: What's the weather like in San Francisco?

**Assistant**: I'll check the current weather in San Francisco for you.

[Uses weather API to fetch real-time data]

Current weather in San Francisco:
- Temperature: 18°C (64°F)
- Conditions: Partly cloudy
- Humidity: 65%
- Wind: 15 km/h W

Would you like a detailed forecast?
